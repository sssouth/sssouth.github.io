经典问题之最长公共前缀

本次来探讨字符串数组的最长公共前缀问题

问题简介：

编写一个函数来查找字符串数组中的最长公共前缀，如果不存在公共前缀，返回””。

问题分析：

1.需要建立一个二维字符串数组，将多个字符串保存起来；此处需要注意，输入字符串数组时,使用的是一层循环
2.由于最长公共前缀的长度不超过最短的字符串，因此，可以找到最短的字符串，减少循环的次数
3.将字符串数组的第一个和第二个字符串的首元素进行比较，若相等，继续进行第一个字符串和第三个字符串的首元素的比较，若相等，继续进行接下来的比较，若不相等，结束比较
4.若第一个元素是公共的元素，则将它保存在一个新的一维字符数组中，按照第二步的方法继续进行比较下一个元素，直至有不相同的元素出现，结束比较
5.输出最长公共前缀

代码实现如下：

以下的功能实现寻找公共前缀：

void Search(char A[][N])
{
    char B[N];
    int i,j,k,min=0,flag=1;
    min=strlen(A[0]);    //min表示字符数组中最短的字符串长度
    for(i=1;i<M;i++)
    {
        if(strlen(A[i])<min)
            min=strlen(A[i]);
    }
    for(i=0;i<min;i++)
    {
        j=0;k=1;flag=1;
        while(k<M)
        {
            if(A[j][i]!=A[k][i])//若第一个数组与第二个数组的首元素不一致，则直接不用再进行比较，flag作为标识
            {
                flag=0;break;
            }
            else    k++;
        }
        if(flag)
            B[i]=A[0][i];//将相等元素存入字符数组B中
        else    break;
    }
    B[i]='\0';//末尾应手动加上字符结束标志'\0'
    printf("最长公共前缀：\"%s\"",B);
}

以下是主函数的代码：

#include <stdio.h>
#include <string.h>
#define M 4
#define N 20
void Search(char A[][N]);
int main(void)
{
    int i;
    char A[M][N];
    printf("请输入字符串数组:\n");
    for(i=0;i<M;i++)    //输入N组字符串(gets()参数为数组名，即数组的首元素地址)
        gets(A[i]);
    Search(A);
    return 0;
}

运用的知识：

1.字符数组的使用
2.以字符数组名为函数参数的使用
