经典问题之爬楼梯

本次的分享是爬楼梯问题

题目简介：

假设你正在爬楼梯，需要n阶你才能到达楼顶。每次你可以爬1或2个台阶，请问有多少种不同的方法可以爬到楼顶?

题目分析：

从题目我们可以获取到一条重要信息：每次爬一阶或两阶。当我们最后一步登顶时，要么爬一阶，要么爬两阶，如果最后一步爬一阶台阶，则前面爬了n-1阶台阶；如果最后一步爬两阶台阶，则前面爬了n-2阶台阶。以此类推，n-1节台阶又可分为最后一步爬一阶或爬两阶···，直至剩下一阶或两阶台阶，我们可以清楚的知道爬一阶台阶只有一种方法，爬两阶台阶有两种方法。因此，我们可以用递归方法求出爬楼梯的总方法数，n=1,n=2即为递归的出口。

代码实现如下：

以下为递归函数的代码：

int count(int n)
{
    if(n == 1)   //函数出口
        return 1;
    if(n == 2)   //函数出口
        return 2;
    return count(n-1) + count(n-2);
  //要么最后一次爬一阶台阶(之前爬了n-1阶台阶)，要么爬两阶台阶(之前爬了n-2阶台阶)
}

以下为主函数代码：

#include <stdio.h>
int main(void)
{
    int n;
    printf("请输入你想要爬的楼梯数:");
    scanf("%d",&n);
    printf("爬%d阶楼梯可以有%d种方法!",n,count(n));
}

以上为对爬楼梯问题的简单分享！！！
