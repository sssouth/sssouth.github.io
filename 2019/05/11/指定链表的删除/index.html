指定链表的删除

本次分享的问题为指定链表的删除

问题简介

给定一个链表，删除链表的倒数第 n 个节点，并且输出删除后的链表。

问题分析

  1. 该问题主要考查链表的有关知识，考查链表的删除以及链表的创建
  2. 该问题得删除操作是倒序删除节点，我们最先想到的方法大多是先记录节点个数，然后找到要删除的节点位置，完成操作；由于链表的节点个数是已知的，所以我们应该找到一种不用计算节点个数就能完成操作的方法，尽可能的优化算法

  3. 若我们创建一个指针变量 P，让 P 指向第 n 个节点后，接着创建另一个指针变量 S，两个指针变量均向后指向，当 P 指向最后一个节点时，S 恰指向要删除的节点，此时只需将 S->next=S->next->next，完成删除

    代码实现如下

    主函数代码如下:

    #include <stdio.h>
#include <stdlib.h>
typedef struct data   //定义结构体
{
    int num;
    struct data * next;
}Data;
void Create(Data * head);   //创建链表
void Print(Data * head);   //打印链表
void Delete(Data * head);  //删除链表

int main(void)
{
    Data * head=(Data * )malloc(sizeof(Data));
    head->next=NULL;
    Create(head);
    Print(head);
    Delete(head);
    return 0;
}

    创建链表的代码如下:

    void Create(Data * head)
{
    char ch;
    Data * temp=head;
    Data * add=(Data * )malloc(sizeof(Data));
    do
    {
        printf("\n请输入节点的数据:");
        scanf("%d",&add->num);
        temp->next=add;
        temp=add;  //temp一直指向尾节点
        add=(Data * )malloc(sizeof(Data));
        printf("\n是否要继续添加节点?(继续数入Y/y)：");
        fflush(stdin);
        ch=getchar();
    }while(ch=='Y' || ch=='y');
    free(add);  //释放申请的空间
    temp->next=NULL;
}

    删除指定节点的代码如下：

    void Delete(Data * head)
{
    int i,n;
    Data * p=head,* s=head;
    printf("\n请输入你想要删除的倒数节点:");
    scanf("%d",&n);
    while(n<=0)  //保证 n 大于0
    {
        printf("\n输入错误,请重新输入:");
        scanf("%d",&n);
    }
    for(i=0;i<n;i++)
    {
        p=p->next;
        if(p==NULL)
        {
            printf("\n输入的删除数据无效!\n");
            exit(1);
        }
    }
    while(p->next)  //找到要删除的节点位置
    {
        p=p->next;
        s=s->next;
    }
    s->next=s->next->next;  //删除节点
    Print(head);
}

打印节点的代码如下:

void Print(Data * head)
{
    Data * temp=head->next;
    while(temp->next!=NULL)
    {
        printf("%d->",temp->num);
        temp=temp->next;
    }
    printf("%d\n",temp->num);
}
